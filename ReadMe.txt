========================================================================
    AOR : Argentum Online Raagh Project Overview
========================================================================
Este proyecto esta pensado como base para una dll que intercepte paquetes del juego
Argentum Online mediante el "hookeo" de funciones del cliente. El proyecto en su estado actual
hookea las funciones de HandleData,SendData y el loop de vb6 para leer los paquetes que el cliente
envia al servidor y viceversa. El proyecto anterior ademas de contener esto contenia el hack completo con 
las funcionalidades implementadas. Por cuestiones de etica se removio el cheat y se dejo solo la base de hookeo
usando MS Detours.
